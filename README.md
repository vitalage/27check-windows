# 27文字チェックツール

原稿が「1行27文字以内」のルールを守っているかチェックするツールです。

---

## 使い方（3ステップ）

**ステップ1**
27check.vbs をデスクトップに置く

**ステップ2**
原稿ファイル（.txt または .md）を 27check.vbs にドラッグ&ドロップ

**ステップ3**
結果ファイルが自動で開きます！

---

## 結果の見方

### OK の場合
```
判定: OK
違反数: 0
問題ありません。
```
→ そのまま提出してOK！

### NG の場合
```
判定: NG
違反数: 3

|行|ワード|文字数|超過|抜粋|
|26|9|35|8|長すぎる文章が...|
```
→ 「行」の番号を見て、該当箇所を修正してください

---

## 表の見方

| 列 | 意味 |
|----|------|
| 行 | 原稿の何行目か |
| ワード | 何番目のワード（3行で1ワード） |
| 文字数 | その行の文字数 |
| 超過 | 27文字を何文字オーバーしているか |
| 抜粋 | 該当行の内容（最初の80文字） |

---

## 動作環境

- Windows 10 / 11
- 何もインストール不要！

---

## 困ったとき

**Q: ダブルクリックしても動かない**
→ ファイルをドラッグ&ドロップしてください

**Q: 結果ファイルが文字化けする**
→ メモ帳で開く場合は「UTF-8」で開いてください
